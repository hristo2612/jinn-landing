## Upcoming surface

- Durable Todos with guarded ownership, lifecycle evidence, linked execution attempts, review policies, and routed approvals.
- Atomic tracked delegation that creates or reuses a Todo, links a child session, and supports caller-scoped idempotency.
- Validated Workflow graphs, durable run evidence, approval gates, cron/poll/webhook/Todo trigger bindings, and authenticated event ingress.
- A built-in session-scoped Jinn MCP server exposing typed company tools for org discovery, sessions, Todos, Workflows, knowledge, and reference search.

## Migration

The `0.26.0` migration introduces the four-block company model and related workspace guidance. Run `jinn migrate` to print and review the composed prompt, then run `jinn migrate --apply` to launch the configured agent and advance the marker after a successful migration.
