# Brand raster sources

`noto-genie-1f9de.png` is the 512px genie (U+1F9DE) from
[Noto Emoji](https://github.com/googlefonts/noto-emoji), pinned to the exact
revision it was taken from:

- **Source file:**
  [`png/512/emoji_u1f9de.png` @ `de7f127a26f7044e5647490af9e3b187d9532703`](https://github.com/googlefonts/noto-emoji/blob/de7f127a26f7044e5647490af9e3b187d9532703/png/512/emoji_u1f9de.png)
- **SHA-256 (committed copy, byte-identical to that revision):**
  `a6f989324e9a74d10e907e7e4166640c4b1db45eaa8f36412671d1fd197c2693`
- **License at that revision:** Apache License 2.0 — the applicable license
  text is redistributed alongside the source as
  [`LICENSE-noto-emoji`](./LICENSE-noto-emoji), and the upstream copyright
  attribution (Apache-2.0 §4(c)) as
  [`AUTHORS-noto-emoji`](./AUTHORS-noto-emoji) — both fetched from the same
  pinned revision. Do not follow the moving `main` branch for provenance:
  newer Noto Emoji revisions relicensed under OFL-1.1, which does not apply
  to this pinned artwork.

It is the source for every shipped raster brand asset (`/favicon-32.png`,
`/favicon-64.png`, `/apple-touch-icon.png`, and the genie in `/og.png`),
regenerated by `scripts/build-brand-assets.ts`. The same Apache-2.0 terms
cover those derivatives.

Apple's emoji artwork is **never** rasterized into this repository — it is
not licensed for redistribution. The SVG favicon and the in-page brand marks
render the raw emoji glyph through the visitor's own platform font instead.
