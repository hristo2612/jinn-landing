# Security policy

## Reporting a vulnerability

Please report security issues through the repository's private security advisory form:

https://github.com/hristo2612/jinn-landing/security/advisories/new

Do not open a public issue containing credentials, private gateway responses, session data, local paths, or reproduction artifacts with personal information.

Include the affected route or file, impact, reproduction steps, and any suggested remediation. You will receive an acknowledgement through GitHub as soon as the report is reviewed.

For vulnerabilities in the Jinn gateway or CLI rather than this website, use the private advisory form in the [Jinn product repository](https://github.com/hristo2612/jinn/security/advisories/new).
